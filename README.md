# 🤖 JARVIS AI Assistant

An advanced AI Voice Assistant built using **Python**, **Tkinter**, **Speech Recognition**, and **Text-to-Speech** technologies.

This project features a futuristic **JARVIS HUD interface** with voice interaction, animations, command execution, and smart assistant capabilities inspired by Iron Man's JARVIS.

---

# 📌 Features

## 🎙️ Voice Recognition

- Recognizes user voice commands
- Converts speech to text using Google Speech Recognition

## 🔊 Text to Speech

- AI voice responses using pyttsx3
- Natural speaking assistant

## 🖥️ Futuristic HUD Interface

# 🚀 Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/Sathvik0728/Jarvis-AI-Assistant.git
```

---

## 2️⃣ Open Project Folder

```bash
cd Jarvis-AI-Assistant
```

---

## 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Run the Project

```bash
python jarvis.py
```

---

# 🧠 How It Works

1. User clicks **ACTIVATE JARVIS**
2. Assistant starts listening
3. Voice converted to text
4. Commands are processed
5. JARVIS responds with speech
6. HUD animations run continuously
